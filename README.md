# WebGPUStudio

A software to teach and learn WebGPU

## Play with it (soon - mid january 2026)

- Visit the website https://virtulab.univ-brest.fr/webgpustudio

## Install localy

1- Download the project

2- In a terminal, go to the project folder

3- Execute: python3 -m http.server 8000

## Use it

4- Open a Web browser to the local url: http://localhost:8000/

5- Click on Open / Charger and choose a program in the Example folder
